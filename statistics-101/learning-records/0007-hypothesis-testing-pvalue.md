# 假設檢定、P 值與女士品茶

建立了現代科學推論的核心邏輯框架：先假設 H₀ 為真（歸謬法），計算 P(觀察到這結果或更極端 | H₀)，若 P 值 < α（通常 0.05）則拒絕 H₀。透過費雪的女士品茶實驗，理解了這個框架的具體運作：8 杯茶、4 奶先 4 茶先的實驗設計，使「瞎猜」的機率可精確計算（超幾何分布），只有 8 杯全對（P = 1/70 ≈ 1.4%）能拒絕 H₀。學會了 P 值的三大誤解：P 值 ≠ H₀ 為真的機率、P > 0.05 ≠ H₀ 為真、P < 0.05 ≠ 效果很大。理解了型一錯誤（偽陽性）與型二錯誤（偽陰性）的取捨——α = 0.05 是保守主義的選擇。

**Implications**：假設檢定的邏輯是科學方法的核心——從臨床試驗到 A/B 測試。P 值的正確理解是統計素養的關鍵指標。
