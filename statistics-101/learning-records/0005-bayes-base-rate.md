# 貝氏定理與基本比率謬誤

學會了將條件機率「反轉」的數學框架：P(H|E) = P(E|H)·P(H) / P(E)。透過罕見疾病篩檢案例，親身體驗了基本比率謬誤——即使檢驗準確率 99%，當疾病盛行率僅 0.1% 時，陽性結果真正患病的機率只有約 9%。關鍵教訓：事前機率（疾病有多罕見）和概似度（檢驗有多準）同等重要，但人類大腦天生忽略前者。同時理解了「幾乎必然」（機率 = 1 但非確定）與無限猴子定理的現實教訓——數學上的可能性 ≠ 實務上的可行性，當期望時間遠超宇宙年齡時，科學家可以合理視為不可能。

**Implications**：貝氏更新思維（信念應隨證據修正）是銜接頻率主義假設檢定與貝氏統計學的橋樑。基本比率謬誤的警覺性直接適用於後續 P 值的正確解讀。
