# 信賴區間的正確詮釋

解開了「95% 信心水準」的真正含義——不是「真實值有 95% 機率落在區間內」（那是貝氏可信區間），而是「如果重複抽樣無限多次，95% 的區間會捕捉到真實參數」。掌握了公式 CI = x̄ ± z* × σ/√n，以及比例資料的樣本數計算：n = (z*)²·p·(1−p) / E²。由此解開了「1068」這個魔術數字——在 p = 0.5、95% 信心水準（z* = 1.96）、誤差 ±3% 的最保守假設下，n = 1067.1 → 1068。最關鍵的洞察：樣本數公式中完全沒有母體大小的項——1068 人對台灣（2300 萬人）和對美國（3 億人）的精確度完全相同。

**Implications**：這是整門入門統計學的終點——CLT → 標準誤 → 信賴區間形成完整的推論鏈。從此能獨立解讀民調、科學研究與數據新聞中的統計主張。
