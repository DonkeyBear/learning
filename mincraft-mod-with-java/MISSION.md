# Mission: Java 生態系職涯轉換

## Why
從前端工程師（3 年 JS/TS）轉向 Java 後端生態系，以 Minecraft MOD 開發（Fabric）作為驅動學習 Java 的趣味 side project，最終目標是掌握 SpringBoot 並勝任 Java 全端/後端工程師職位。

## Success looks like
- 能獨立使用 Java + SpringBoot 搭建一個完整的 REST API 後端服務
- 能獨立開發並發布一個可玩的 Minecraft Fabric MOD
- 能在 Java 技術面試中自信地討論語言特性、JVM 原理和後端架構
- 具備閱讀並理解開源 Java 專案原始碼的能力

## Constraints
- 每週投入時間不固定，先以靈活節奏開始，邊學邊調整
- 偏好 Fabric 作為 MOD 開發環境（但不排斥其他選項如果理由充分）
- 課件需使用深色模式（dark mode）
- 已有 OOP 基礎和理解，不需要從零教程式設計概念
- 已有前後端分離的架構理解（Python FastAPI 經驗）

## Out of scope
- 前端框架或 UI 開發（已具備足夠經驗）
- Python 生態系深入內容
- 遊戲設計理論（除非直接服務於 MOD 開發）
- Android/Kotlin 開發（專注於伺服器端 Java）
