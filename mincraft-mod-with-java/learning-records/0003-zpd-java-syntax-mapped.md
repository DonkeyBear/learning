# ZPD: Java Syntax Mapped to TS/Python

Recognized that since the user is already proficient in TS and Python, starting from scratch with basic programming concepts (loops, what is a variable) is a waste of their working memory and motivation.

Instead, the most effective pedagogical approach for their Zone of Proximal Development (ZPD) is **Knowledge Mapping**: presenting Java's syntax and paradigms directly mapped against TypeScript and Python.

**Evidence**: User has 3 years of JS/TS experience and Python FastAPI experience.
**Action taken**: Lesson 0002 focuses entirely on the differences: strict typing, everything-in-a-class rule, access modifiers vs getters/setters, and the crucial difference between fixed Arrays and dynamic `ArrayList`. Created a quick-reference cheatsheet.
**Future Implications**: Continue skipping basic programming logic. Introduce Java-specific features (Streams, Lambdas, Optional) by comparing them to JS Array methods and TS null-checking.
