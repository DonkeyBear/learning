# Java + Minecraft MOD + SpringBoot Resources

## Knowledge

### Java Fundamentals
- [Book: _Effective Java_ (3rd Edition) — Joshua Bloch](https://www.oreilly.com/library/view/effective-java/9780134686097/)
  Java 聖經級讀物。Use for: 深入理解 Java 的最佳實踐、設計模式、語言陷阱。適合有基礎後精讀。
- [Official: Java Tutorials — Oracle](https://docs.oracle.com/javase/tutorial/)
  官方教學。Use for: 查閱語言特性的權威說明。

### Minecraft MOD Development (Fabric)
- [Official: Fabric Wiki](https://fabricmc.net/wiki/start)
  Fabric 官方文件，最權威的入門資源。Use for: 環境設定、基礎 MOD 結構、API 參考。
- [Official: Fabric API Documentation](https://fabricmc.net/develop/)
  開發者文件。Use for: API 使用方式和最新更新。

### SpringBoot
- [Official: Spring Boot Reference Documentation](https://docs.spring.io/spring-boot/documentation.html)
  官方參考文獻。Use for: 查閱設定、annotation、自動配置的權威說明。
- [Book: _Spring in Action_ (6th Edition) — Craig Walls](https://www.manning.com/books/spring-in-action-sixth-edition)
  Spring 生態系經典入門書。Use for: 系統性學習 Spring/SpringBoot 核心概念。
