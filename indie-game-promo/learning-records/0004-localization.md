# 商店頁面在地化的 ROi 遠高於多數行銷手段

用戶學習了 Steam 語系推薦過濾的硬性機制，核心洞察：
- Steam 演算法不會推薦不支援該語系的遊戲——不是排後面，是直接跳過
- 階段式在地化：商店頁面文本（$200-800/語系）→ 遊戲 UI → 敘事文本。第一階段就能解鎖全語系推薦曝光
- 五大優先語系：簡中 → 英文 → 俄語 → 德語 → 日語。預算極限時優先英+俄
- 截圖中嵌入宣傳文字會造成多語系維護災難——最佳解法是不在截圖中放文字
- 2026 年中國 FMV 遊戲成功打入西方市場證明了：文化背景不是障礙，缺乏在地化才是

**Implications**: 用戶的商店頁面一旦建立，應立即規劃在地化預算。後續課程可深入願望清單漏斗、新品節戰術，這些都需要商店頁面各語系先就位。
