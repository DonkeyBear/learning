# Teaching Notes

## User Preferences
- **Language**: 簡體中文為主，可混用英文專有名詞
- **Pace**: 系統性深入，每個主題要講透，不急
- **Style**: 偏好數據驅動、實證導向的內容（呼應 Gemini report 的風格）
- **Interactivity**: 偏好可操作的練習和實際應用到自己專案上的作業
- **Theme**: 深色模式（dark mode）。所有課件與參考文件統一使用暗色背景配色

## Project Context
- 遊戲類型：待確認（首次課程中會引導用戶進行類型市場適配度分析）
- 開發階段：底層 framework 接近完成，即將開始關卡內容填充
- 團隊規模：個人/小團隊

## Working Notes
- 2026-06-13: 初始化工作區。用戶提供了 Gemini deep research 報告（2026 Steam 獨立遊戲市場分析），涵蓋演算法變革、新品節數據、轉化率、定價策略等。報告品質高，可作為多堂課程的基礎素材。
- 第一堂課的方向：類型市場適配度（Genre-Market Fit）——因為用戶專案尚未進入大量內容生產，現在是評估市場定位的最佳時機。
