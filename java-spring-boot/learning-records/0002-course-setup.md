# Course Setup: Spring Boot 學習路徑確立

完成 8 週課程規劃，涵蓋 24 堂課（12 理論 + 12 實作），從 Java 語法基礎開始，逐步進入 Spring Boot 核心（DI/IOC/Bean 生命週期）、資料庫整合、並行處理，最終以完整 REST API 專案收尾。User 選擇邊學 Spring Boot 邊補 Java（選項 B），理論課與實作課分開設計以利通勤學習，IDE 使用 IntelliJ Community Edition。

**Implications**：
- 每堂理論課須可獨立在手機上閱讀，不依賴實作步驟
- 實作課需包含 IntelliJ 操作指引
- 對比式教學（Java↔TS/Python, Spring Boot↔FastAPI）貫穿全課程
