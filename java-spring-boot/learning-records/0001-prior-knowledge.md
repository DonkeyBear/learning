# Prior Knowledge: 前端工程師轉後端

User 具備三年前端工程師經驗，熟悉 JavaScript、TypeScript、Vue、React，並有使用 Python FastAPI 撰寫簡易後端的經驗。對於 HTTP 協定、REST API 設計、JSON 處理、非同步程式設計（Promise/async-await）已有實戰理解。

**為何重要**：User 並非程式新手，而是有經驗的開發者在擴展技能棧。可以跳過基礎程式設計概念（變數、迴圈、函式、HTTP），直接聚焦 Java/Spring Boot 特有的思維模式與設計典範。Java 的靜態型別、OOP 強制性、以及 Spring 的 DI 容器是本課程的核心認知轉換點。

**Evidence**：User 自行陳述三年前端資歷與 FastAPI 經驗。具體深度待第一堂課驗證。

**Implications**：
- 每堂課應對比 JS/Python 中對應概念，降低認知轉換成本
- TypeScript 經驗代表已熟悉靜態型別思維，Java 型別系統可快速帶過
- FastAPI 的 Depends() 是理解 Spring DI 的絕佳跳板
