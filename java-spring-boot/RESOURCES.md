# Spring Boot 學習資源

## Knowledge

- [Spring Boot 官方文件](https://docs.spring.io/spring-boot/documentation.html)
  最權威的一手參考資料。Use for: 所有 Spring Boot 概念的權威定義、設定參數查詢。

- [Baeldung — Spring Boot Tutorials](https://www.baeldung.com/spring-boot)
  社群公認最高品質的 Spring 教學網站，涵蓋幾乎所有 Spring 生態系主題。Use for: DI/IOC、Bean 生命週期、JPA、REST API 實作等深度教學文章。

- [Spring Framework 官方文件](https://docs.spring.io/spring-framework/reference/)
  Spring Framework 核心概念（DI container、AOP）的權威來源。Use for: DI/IOC 底層機制、Bean 生命週期各階段細節。

- [Java 17 API 文件](https://docs.oracle.com/en/java/javase/17/docs/api/)
  Java 標準庫的官方文件。Use for: Stream API、Collections、Concurrency 等標準庫查詢。

- [Mkyong — Spring Boot](https://mkyong.com/spring-boot/)
  簡潔直接的 Spring Boot 範例教學。Use for: 快速查找特定功能的實作範例。

## Wisdom (Communities)

- [r/learnjava](https://reddit.com/r/learnjava)
  針對 Java 初學者的友善社群，問題回覆品質高。Use for: Java 語言層面的疑問。

- [Stack Overflow — Spring Boot](https://stackoverflow.com/questions/tagged/spring-boot)
  Spring Boot 相關問題的主要聚集地。Use for: 除錯與特定問題解答。

## Gaps
- Spring Boot 繁體中文教學資源較少，大部分為簡體或英文 — 課程本身將填補此缺口
- 以 JS/TS/Python 開發者視角切入的 Spring Boot 課程稀缺 — 本課程將以此為定位
