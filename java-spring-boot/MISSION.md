# Mission: Java Spring Boot 後端開發

## Why
從三年前端工程師轉職為全端或後端工程師，需要在 1-2 個月內具備 Spring Boot 實作能力並通過後端面試。已有 FastAPI 簡易後端經驗，但需要補齊 Java 語言基礎及 Spring Boot 生態系知識，特別是面試必考的 DI、IOC、Bean 生命週期、Java Stream、執行緒安全等核心概念。

## Success looks like
- 能獨立用 Spring Boot 搭建一個完整的 REST API 專案（含資料庫、分層架構）
- 能清楚解釋 DI/IOC 的設計原理與 Spring 的實作方式，足以通過面試白板題
- 能說明 Bean 生命週期各階段及其應用場景
- 能使用 Java Stream API 進行資料處理，並理解執行緒安全的基礎概念
- 具備用 IntelliJ Community Edition 進行 Java/Spring Boot 開發的生產力

## Constraints
- 每週 10-15 小時學習時間
- 部分時間在通勤（手機閱讀），理論課與實作課需分開設計
- IDE 使用 IntelliJ Community Edition（免費版）
- 目前完全不會 Java，需邊學 Spring Boot 邊補 Java 基礎
- 1-2 個月後開始面試

## Out of scope
- Microservices / Spring Cloud（初學階段暫不觸及）
- Kotlin 或其他 JVM 語言
- DevOps / CI/CD / Docker 部署（面試後再補）
- React/Vue 前端整合（已具備前端能力，專注後端）
