Stream API 實戰練習

這堂課你會拿到一份模擬的員工資料集，用 T03 學到的 Stream API 完成 10 道練習題。題目從簡單到進階，涵蓋日常開發和面試中常見的資料處理模式。

🎯 練習方式 每道題目給你「需求描述」和「預期輸出」，你要在 IntelliJ 中自己寫出 Stream 管線。建議先自己嘗試，卡住了再看答案。每題的答案區用 <details> 摺疊，不要偷看。

Step 1：建立練習環境

在 IntelliJ 的 java-practice 專案中，建立 practice/StreamLab.java：

package practice;

import java.util.*;
import java.util.stream.*;

public class StreamLab {

    // ====== 資料集：員工列表 ======
    public static final List<Employee> EMPLOYEES = List.of(
        new Employee("Alice",   "Engineering", 95000,  List.of("Java", "Spring", "AWS")),
        new Employee("Bob",     "Engineering", 82000,  List.of("Python", "Django", "Docker")),
        new Employee("Charlie", "Engineering", 78000,  List.of("Java", "Kotlin")),
        new Employee("Diana",   "Product",     88000,  List.of("SQL", "Excel", "Tableau")),
        new Employee("Eve",     "Product",     92000,  List.of("SQL", "Python", "Figma")),
        new Employee("Frank",   "HR",          65000,  List.of("Excel", "Communication")),
        new Employee("Grace",   "HR",          67000,  List.of("Excel", "Payroll")),
        new Employee("Henry",   "Engineering", 105000, List.of("Java", "Spring", "Kubernetes", "AWS")),
        new Employee("Ivy",     "Product",     91000,  List.of("SQL", "A/B Testing", "Figma")),
        new Employee("Jack",    "Engineering", 72000,  List.of("Java", "React"))
    );

    // ====== Employee Record ======
    public record Employee(
        String name,
        String department,
        int salary,
        List<String> skills
    ) {
        public String summary() {
            return name + " | " + department + " | $" + salary;
        }
    }

    public static void main(String[] args) {
        System.out.println("=== Stream API 練習 ===\n");

        // 在下面逐題寫你的 Stream 答案
        // ex1_filterHighEarners();
        // ex2_getNames();
        // ...
    }
}
💡 快速鍵提醒 在 main 中寫練習時，用 sout + Tab → System.out.println()；用 EMPLOYEES.stream() 開頭，IntelliJ 會自動提示 Stream 方法。

練習題

練習 1：過濾高薪者 🔹

找出薪水 ≥ 90000 的員工，印出他們的 summary()。

點擊展開答案 
EMPLOYEES.stream()
    .filter(e -> e.salary() >= 90000)
    .map(Employee::summary)
    .forEach(System.out::println);

// 輸出：
// Alice | Engineering | $95000
// Eve | Product | $92000
// Henry | Engineering | $105000
// Ivy | Product | $91000

練習 2：取出不重複的部門 🔹

列出所有部門名稱（不重複），按字母排序。

點擊展開答案 
var depts = EMPLOYEES.stream()
    .map(Employee::department)
    .distinct()
    .sorted()
    .toList();

System.out.println(depts);
// [Engineering, HR, Product]

練習 3：薪資統計 🔹

計算所有員工的平均薪資、最高薪資、最低薪資。

點擊展開答案 
double avg = EMPLOYEES.stream()
    .mapToInt(Employee::salary)
    .average().orElse(0);

int max = EMPLOYEES.stream()
    .mapToInt(Employee::salary)
    .max().orElse(0);

int min = EMPLOYEES.stream()
    .mapToInt(Employee::salary)
    .min().orElse(0);

System.out.println("平均: $" + avg);   // 平均: $83500.0
System.out.println("最高: $" + max);   // 最高: $105000
System.out.println("最低: $" + min);   // 最低: $65000

練習 4：按薪資排序取前 N 🔹🔹

找出薪資最高的前 3 名員工，依薪資由高到低排列。

點擊展開答案 
var top3 = EMPLOYEES.stream()
    .sorted(Comparator.comparingInt(Employee::salary).reversed())
    .limit(3)
    .toList();

top3.forEach(e -> System.out.println(e.summary()));
// Henry | Engineering | $105000
// Alice | Engineering | $95000
// Eve | Product | $92000

練習 5：分組統計 — groupingBy 🔹🔹🔥

按部門分組，計算各部門的員工人數和平均薪資。

點擊展開答案 
// 各部門人數
var countByDept = EMPLOYEES.stream()
    .collect(Collectors.groupingBy(
        Employee::department,
        Collectors.counting()
    ));
System.out.println("人數: " + countByDept);

// 各部門平均薪資
var avgByDept = EMPLOYEES.stream()
    .collect(Collectors.groupingBy(
        Employee::department,
        Collectors.averagingInt(Employee::salary)
    ));
System.out.println("平均薪資: " + avgByDept);

// 輸出：
// 人數: {Product=3, HR=2, Engineering=5}
// 平均薪資: {Product=90333.33..., HR=66000.0, Engineering=86400.0}

練習 6：收集成 Map 🔹🔹

將員工列表轉成 Map<String, Integer>：Key = 姓名，Value = 薪資。

點擊展開答案 
Map<String, Integer> nameToSalary = EMPLOYEES.stream()
    .collect(Collectors.toMap(
        Employee::name,
        Employee::salary
    ));

System.out.println(nameToSalary);
// {Alice=95000, Bob=82000, Charlie=78000, Diana=88000,
//  Eve=92000, Frank=65000, Grace=67000, Henry=105000,
//  Ivy=91000, Jack=72000}

练习 7：flatMap 攤平技能 🔹🔹

收集所有不重複的技能，按字母排序。

點擊展開答案 
var allSkills = EMPLOYEES.stream()
    .flatMap(e -> e.skills().stream())
    .distinct()
    .sorted()
    .toList();

System.out.println(allSkills);
// [A/B Testing, AWS, Communication, Django, Docker,
//  Excel, Figma, Java, Kotlin, Kubernetes, Payroll,
//  Python, React, SQL, Spring, Tableau]

練習 8：anyMatch / allMatch 🔹🔹

(1) 是否有任何員工薪資超過 100000？
(2) 是否所有員工薪資都超過 60000？
(3) 工程部門中是否有人會 Kotlin？

點擊展開答案 
boolean hasHighEarner = EMPLOYEES.stream()
    .anyMatch(e -> e.salary() > 100000);
System.out.println("有薪資>100k: " + hasHighEarner);  // true

boolean allAbove60k = EMPLOYEES.stream()
    .allMatch(e -> e.salary() > 60000);
System.out.println("全員>60k: " + allAbove60k);  // true

boolean engHasKotlin = EMPLOYEES.stream()
    .filter(e -> "Engineering".equals(e.department()))
    .flatMap(e -> e.skills().stream())
    .anyMatch("Kotlin"::equals);
System.out.println("工程部有人會Kotlin: " + engHasKotlin);  // true

練習 9：partitioningBy 二分法 🔹🔹🔹

將員工按「薪資是否 ≥ 90000」分成兩組，並列出各組的人名。

點擊展開答案 
var partitioned = EMPLOYEES.stream()
    .collect(Collectors.partitioningBy(
        e -> e.salary() >= 90000,
        Collectors.mapping(Employee::name, Collectors.toList())
    ));

System.out.println("高薪組: " + partitioned.get(true));
System.out.println("非高薪組: " + partitioned.get(false));
// 高薪組: [Alice, Eve, Henry, Ivy]
// 非高薪組: [Bob, Charlie, Diana, Frank, Grace, Jack]

練習 10：綜合挑戰 🔹🔹🔹🔥

找出技能數量最多的部門中，薪資最高的那位員工的姓名和薪資。

（提示：先用 groupingBy 算各部門總技能數 → 找最大 → 再從該部門找薪資最高者）

點擊展開答案 
// Step 1: 各部門的總技能數
var skillCountByDept = EMPLOYEES.stream()
    .collect(Collectors.groupingBy(
        Employee::department,
        Collectors.flatMapping(
            e -> e.skills().stream(),
            Collectors.counting()
        )
    ));
System.out.println("各部門技能數: " + skillCountByDept);

// Step 2: 找技能數最多的部門
String topDept = skillCountByDept.entrySet().stream()
    .max(Map.Entry.comparingByValue())
    .map(Map.Entry::getKey)
    .orElse("");

System.out.println("技能最多部門: " + topDept);  // Engineering

// Step 3: 該部門薪資最高的員工
var topEarner = EMPLOYEES.stream()
    .filter(e -> topDept.equals(e.department()))
    .max(Comparator.comparingInt(Employee::salary))
    .orElseThrow();

System.out.println("結果: " + topEarner.name() + " $" + topEarner.salary());
// 結果: Henry $105000

例如：「每個部門中技能種類最多的員工」「把員工依技能數分三級：初階(1-2)、中階(3)、高階(4+)」

課後檢查清單

  • filter + map + toList 基本管線流暢
  • sorted + limit 取前 N 名無誤
  • groupingBy 搭配 counting / averaging 能寫出來
  • flatMap 攤平巢狀集合熟練
  • anyMatch / allMatch / noneMatch 語意清楚
  • partitioningBy 二分法會用
  • 練習 10 綜合挑戰能不看答案獨立完成

📌 這堂課的 Stream 操作地圖

filter → map → toList最常見的基本管線
sorted + limitTop-N 查詢
groupingBy + averaging/counting分組聚合 = SQL GROUP BY
flatMap + distinct攤平巢狀 + 去重
collect(toMap)List → Map 轉換
anyMatch / allMatch條件判斷 = JS some/every
partitioningBy二分法分組
