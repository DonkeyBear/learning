IntelliJ 環境設定 + 第一個 Java 專案

這堂課會帶你從零開始安裝 IntelliJ Community Edition，建立第一個 Java 專案，並實際跑一次 T01 教過的語法。

🎯 這堂課的目標 不是寫出什麼厲害的程式，而是讓你的開發環境就緒。後續每一堂實作課都會在這個基礎上疊加，所以確保每一步都走完。

1. 安裝 IntelliJ IDEA Community Edition

ℹ️ Community Edition 是完全免費的，不需要付費授權。它對 Spring Boot 開發完全夠用。付費的 Ultimate 版多了內建 Spring 支援，但我們會用手動設定來達成相同效果。

步驟

  1. 前往 JetBrains 官網下載頁
  2. 選擇 IntelliJ IDEA Community Edition（不是 Ultimate）
  3. 下載 Windows .exe 安裝檔
  4. 執行安裝程式：
    • 全部用預設選項即可
    • 「Create Desktop Shortcut」→ 建議勾選
    • 「Add "Open Folder as Project"」→ 建議勾選
    • 「Add bin to PATH」→ 可選（讓終端機可以直接用 idea 指令開啟）
  5. 安裝完成後啟動 IntelliJ
💡 順帶一提：Toolbox App JetBrains 也有提供 Toolbox App，可以統一管理所有 JetBrains IDE 的版本更新。如果你預計未來也會用 WebStorm、DataGrip 等工具，推薦安裝。對本課程而言直接用 standalone 安裝就夠了。

2. 建立第一個 Java 專案

啟動 IntelliJ 後：

  1. 點擊 New Project
  2. 設定以下選項：
    • Name：java-practice
    • Location：選一個你方便找到的資料夾
    • Language：Java
    • Build system：IntelliJ（先不用 Gradle 或 Maven，下一堂再來）
    • JDK：如果顯示 "No SDK"，點擊下拉選單 → Download JDK → 選 Amazon Corretto 21（或 Oracle OpenJDK 21）→ Download
  3. 點擊 Create
⚠️ JDK 版本說明 建議選 JDK 21（LTS，長期支援版本）。Spring Boot 3.x 最低要求 JDK 17，選 21 可以涵蓋你未來所有 Spring Boot 開發需求。Corretto 是 Amazon 維護的免費 OpenJDK 發行版，穩定且無授權問題。

建立完成後，你會看到這樣的專案結構：

java-practice/
├── .idea/            ← IntelliJ 設定檔（不要手動修改）
├── src/
│   └── Main.java     ← IntelliJ 自動產生的主程式
└── java-practice.iml  ← 模組設定檔

3. 你的第一支 Java 程式

IntelliJ 可能已經幫你產生了一個 Main.java。確認內容如下（如果沒有，在 src 資料夾上按右鍵 → New → Java Class → 命名為 Main）：

public class Main {
    public static void main(String[] args) {
        System.out.println("Hello, Spring Boot!");
    }
}

按一下 main 方法左側的 ▶ 綠色三角形，選擇 Run 'Main.main()'。

下方的 Run 視窗 應該顯示：

Hello, Spring Boot!

Process finished with exit code 0

🎉 恭喜！你的 Java 環境就緒了。

💡 IntelliJ 快速鍵備忘
Alt+Enter萬能修復鍵（import、修正錯誤、建議）
Ctrl+Shift+F10執行目前檔案
Shift+F10重新執行上一次的 Run
Ctrl+Space自動完成
Ctrl+/單行註解切換
psvm + Tab快速產生 public static void main
sout + Tab快速產生 System.out.println()

4. 動手練習：複習 T01 的語法

把 Main.java 的內容全部換掉，改寫成以下程式，實際跑一次 T01 學到的語法：

import java.util.List;
import java.util.ArrayList;

public class Main {

    // 練習 1：基本型別與物件型別
    public static void main(String[] args) {
        // 基本型別（小寫）
        int score = 85;
        double price = 29.99;
        boolean isActive = true;

        // 物件型別（大寫）
        String playerName = "Alice";
        Integer wrappedScore = score;  // int 自動裝箱成 Integer

        // 練習 2：動態列表（List）
        List<String> inventory = new ArrayList<>();
        inventory.add("Potion");
        inventory.add("Elixir");
        inventory.add("Phoenix Down");

        // 練習 3：for-each 迴圈
        System.out.println("=== 背包內容 ===");
        for (String item : inventory) {
            System.out.println("- " + item);
        }

        // 練習 4：字串比較（用 equals，不是 ==）
        String answer = "yes";
        if (answer.equals("yes")) {
            System.out.println("你選擇了 yes");
        }

        // 練習 5：基本輸出
        System.out.println("玩家：" + playerName);
        System.out.println("分數：" + score);
        System.out.println("背包大小：" + inventory.size());
    }
}

按 Ctrl+Shift+F10（或點綠色 ▶）執行。你應該看到：

=== 背包內容 ===
- Potion
- Elixir
- Phoenix Down
你選擇了 yes
玩家：Alice
分數：85
背包大小：3
⚠️ 如果看到紅色波浪線 把游標移到紅字上，按 Alt+Enter，IntelliJ 會建議 Import class。選擇它，IntelliJ 就會自動在檔案頂部加上 import java.util.List; 等必要的 import 陳述式。這是你在 Java 開發中最常用的快捷鍵，沒有之一。

5. IntelliJ 基礎操作練習

花 5 分鐘熟悉以下操作，它們會在你每天的開發中反覆出現：

  1. 建立新 class：在 src 目錄上按右鍵 → New → Java Class → 輸入 Item
  2. 在 Item.java 中寫入以下內容：
public class Item {
    private String name;
    private int quantity;

    // 建構子
    public Item(String name, int quantity) {
        this.name = name;
        this.quantity = quantity;
    }

    // Getter（Alt+Insert 可以自動產生）
    public String getName() {
        return name;
    }

    public int getQuantity() {
        return quantity;
    }
}
  1. 回到 Main.java，在 main 方法中加入：
// 使用自訂的 Item class
List<Item> shopItems = new ArrayList<>();
shopItems.add(new Item("Sword", 1));
shopItems.add(new Item("Shield", 2));

for (Item item : shopItems) {
    System.out.println(item.getName() + " x" + item.getQuantity());
}
  1. 執行，確認輸出：
Sword x1
Shield x2
💡 Getter/Setter 自動產生
  1. 在 class 內的空白處按 Alt+Insert
  2. 選擇 Getter（或 Setter）
  3. 勾選你要的屬性 → OK
IntelliJ 會自動產生正確的 getter/setter 方法。Spring Boot 大量依賴 getter/setter，記住這個快捷鍵會省下大量時間。
