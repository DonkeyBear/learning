# Mission: React Native Mastery for Mobile Engineering Transition

## Why
我要從前端工程師轉職為 Mobile App 工程師。我已有 React 基礎和 Expo prototype 經驗，但對 RN 底層架構（bridge、native module、threading model）理解不深，無法獨立做出具備求職競爭力的 APP。我需要系統補足從架構理解、生產級開發模式、到雙平台發布的完整能力鏈。

## Success looks like
- 能解釋 RN runtime 架構（JS thread → bridge → native thread）以及新舊架構的差異
- 能獨立從零搭建一個 Expo managed workflow 專案，包含 navigation、state management、API 串接
- 能撰寫原生模組（native module）橋接 iOS/Android 平台功能
- 能將 APP 成功上架到 App Store 和 Google Play（或至少完成上架準備）
- 擁有一個可供求職展示的完整商業應用作品（電商或社交類）

## Constraints
- 時間線：1-2 個月密集學習（利用現有 React 經驗加速）
- 雙平台（iOS + Android）via Expo
- 先打基礎，再往商業應用靠齊
- 需要繁體中文資源或至少英文資源無障礙

## Out of scope
- Flutter、SwiftUI、Jetpack Compose 等非 RN 技術（除非作為對比理解）
- 遊戲開發（Unity、react-native-game-engine）
- 深度後端架構（仍會觸及 BFF/Firebase 但非主軸）
- React Web 開發（已具備的知識，不做重複教學）
