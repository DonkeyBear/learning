# React Native Resources

## Knowledge

- [React Native Official Docs](https://reactnative.dev/docs/getting-started)
  核心權威來源，React Native 團隊維護。版本跟隨 0.76+（New Architecture 預設啟用）。Use for: 所有基礎概念、API 參考、架構說明。Start with "Learn the Basics" tutorial.

- [Expo Official Docs](https://docs.expo.dev/)
  Expo 生態系統的完整文件，包含 EAS Build、EAS Submit、EAS Update、dev client 等。Use for: 專案初始化、建置部署、OTA 更新、config plugin 撰寫。

- [React Native New Architecture Blog Post (Oct 2024)](https://reactnative.dev/blog/2024/10/23/the-new-architecture-is-here)
  官方宣布 New Architecture（JSI、Fabric、TurboModules、Codegen）從 0.76 開始預設啟用的部落格文章。Use for: 理解新舊架構差異、遷移考量。

- [React Native Architecture: Fabric Renderer](https://reactnative.dev/architecture/fabric-renderer)
  官方架構文件，詳細說明 Fabric 渲染器的運作原理。Use for: 深入理解 Shadow Tree、layout、mounting pipeline。

- [React Navigation Docs](https://reactnavigation.org/docs/getting-started)
  RN 生態中事實標準的導航庫。Use for: stack/tab/drawer navigator 設定、deep linking、navigation 與 type safety。

- [Book: _React Native in Action_ — Nader Dabit (Manning)](https://www.manning.com/books/react-native-in-action)
  評價極高的 RN 實戰書籍，涵蓋從基礎到原生模組的完整範疇。雖然出版年份較早，但核心概念（navigation、state、native modules）仍然有效。Use for: 結構化學習路徑、實戰範例。

- [YouTube: "React Native Tutorial for Beginners" — Academind](https://www.youtube.com/@academind)
  高品質影片教學，適合視覺化學習者。Use for: 概念視覺化、跟著做專案。

## Wisdom (Communities)

- [Reactiflux Discord](https://www.reactiflux.com/)
  最大的 React/RN 即時聊天社群（18,000+ 人），有專屬 #react-native 頻道。Use for: 即時問答、疑難排解、與核心團隊 AMA 互動。

- [r/reactnative (Reddit)](https://reddit.com/r/reactnative)
  活躍的 RN 討論區。Use for: 工具討論、經驗分享、業界趨勢。

- [Stack Overflow — react-native tag](https://stackoverflow.com/questions/tagged/react-native)
  150,000+ 問題，最適合結構化技術提問。Use for: 具體 code-level 問題、錯誤訊息排查。

- [GitHub: react-native-community/discussions-and-proposals](https://github.com/react-native-community/discussions-and-proposals)
  RN 社群的功能提案與架構討論。Use for: 理解設計決策、追蹤未來方向。
