# 已具備的後端基礎能力

使用者已能獨立使用 Python + FastAPI 建立 REST API，透過 Firebase 進行 CRUD 操作，並以 uvicorn 部署到 VM 上。這表示使用者已理解：路由（routing）、HTTP methods（GET/POST/PUT/DELETE）、request/response 模型、基本的部署流程。

**Evidence:** 使用者自述已完成上述開發與部署。
**Implications:** 不需要從零開始教 HTTP 或 FastAPI。課程應從「已能寫出 API 但不知道 request 中間經過了什麼」這個位置出發，第一課適合講 HTTP request 的完整生命週期與中間件（middleware）的位置。
