# 後端基礎架構 Resources

## Knowledge

- [MDN Web Docs: HTTP](https://developer.mozilla.org/en-US/docs/Web/HTTP)
  HTTP 協議的權威參考。Use for: HTTP headers、methods、status codes、CORS 的標準定義。

- [FastAPI 官方文件: Middleware](https://fastapi.tiangolo.com/tutorial/middleware/)
  FastAPI 的中間件機制，直接對應 CORS、proxy headers 等概念的實作。Use for: 把抽象概念轉換為實際程式碼。

- [NGINX 官方入門指南](https://nginx.org/en/docs/beginners_guide.html)
  反向代理與靜態檔案服務的事實標準。Use for: 理解 reverse proxy 的配置與運作。

- [OWASP Top 10](https://owasp.org/www-project-top-ten/)
  Web 應用程式十大安全風險的權威清單。Use for: 資安課程的主要參考架構。

- [JWT.io Introduction](https://jwt.io/introduction)
  JSON Web Token 的互動式說明。Use for: 理解 token-based authentication 的結構與驗證流程。

- [Cloudflare Learning Center: What is an API Gateway?](https://www.cloudflare.com/learning/security/api/what-is-an-api-gateway/)
  以圖解方式解釋 API Gateway 的角色。Use for: 第一堂網關課程的引導閱讀。

- [《Web Application Security: Exploitation and Countermeasures for Modern Web Applications》— Andrew Hoffman (O'Reilly)](https://www.oreilly.com/library/view/web-application-security/9781492053101/)
  現代 Web 安全實務書籍，涵蓋 OWASP Top 10 的防禦策略。Use for: 資安課程的深度參考。

## Wisdom (Communities)

- [r/experienceddevs](https://reddit.com/r/experienceddevs)
  有經驗的開發者社群，討論架構決策與面試準備。Use for: 面試問題實戰討論、架構取捨的現實案例。

- [FastAPI 官方 Discord](https://discord.com/invite/VQjSZaeJmf)
  FastAPI 社群即時討論。Use for: 部署與中介層的實務問題。
